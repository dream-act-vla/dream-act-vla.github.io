# DreamActVLA — Project Report Website

A blog-style project report page for **DreamActVLA: Temporally Aligned Asynchronous Inference for Vision-Language-Action Models**.

A single static HTML page — open `index.html` directly in a browser, or view it hosted on [GitHub Pages](https://pages.github.com/).

## File Structure

```
dream-act-vla/
├── index.html           # Main content
├── assets/
│   ├── css/
│   │   └── style.css    # Styling
│   └── img/              # Figures, images, and videos
└── README.md
```

## Preview Locally

```bash
open index.html
```

No server, build step, or dependencies required.
